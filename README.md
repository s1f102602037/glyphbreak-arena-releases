# GLYPHBREAK ARENA — Official Releases

This repository is used only to distribute official Windows builds and update
metadata. Source code, Unity project caches, credentials, and service secrets
are not published here.

Download the newest ZIP from the GitHub Releases page, extract the complete
folder, and run `GlyphbreakArena.exe`.

Version 1.2 adds 2–4 player online lobbies, per-player fighter and team
selection, free-for-all/team victory rules, and an in-game update checker.

GLYPHBREAK ARENA is an independent project and is not affiliated with or
endorsed by any other game publisher or platform holder.
